# Oral–Gut–Liver Microbiome Modelling

Code for the study:

“Integrative host–microbiome modelling uncovers the implication of oral–gut translocation in advanced cirrhosis”

## Structure

- `Figure 1/1_reactotypes/` — reactotype analysis and OGMD calculation  
- `Figure 1/2_bacteria_single_modelling/` — single-species GEM simulations  
- `Figure 2/1_bacterial_community_modelling/` — community modelling  
- `Figure 2/2_host_simulations/` — host metabolic simulations  

## Requirements

- R  
- MATLAB (COBRA Toolbox, MIGRENE toolbox)
